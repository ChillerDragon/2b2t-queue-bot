# 2b2t-queue-bot

A simple js bot that keeps up the connection. You can run this on your vps.
And then connect with your main client once the bot got far in the queue.

```
$ node src/index.js
◇ injected env (2) from .env // tip: ⌘ suppress logs { quiet: true }
Using minecraft account 'ZillyHuhn'

position in queue: null
position in queue: null
position in queue: 509
position in queue: 509
position in queue: 509
position in queue: 509
position in queue: 509
```

Once the counter approaches 0 you can just turn off the idle client and switch to your real client.
